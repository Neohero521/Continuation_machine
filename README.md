# 彩云小梦风格小说续写 - SillyTavern 扩展
> 复刻彩云小梦核心体验的SillyTavern第三方小说创作扩展，专为网络小说作者打造，集成AI续写、多分支创作、文风定制等全流程创作功能，开箱即用。

## 📦 插件信息
- **插件名称**: 彩云小梦风格小说续写
- **当前版本**: v1.3.0
- **插件作者**: Neohero521
- **项目仓库**: [https://github.com/Neohero521/Continuation_machine](https://github.com/Neohero521/Continuation_machine)
- **兼容环境**: SillyTavern 1.12.0+ 全版本

## ✨ 核心功能特性
### 核心创作能力
- **多分支续写**: 单次生成3条不同剧情走向的续写分支，自由选择最优内容
- **光标精准续写**: 严格从光标所在位置开始续写，开头零空格、零换行，无缝衔接原文
- **智能自动分段**: 续写内容支持根据剧情节奏自动合理分段，保留原文格式的同时提升阅读体验
- **全功能创作工具**: 内置续写、扩写、缩写、改写、定向续写5大核心创作功能
- **多文风适配**: 预设脑洞大开、细节狂魔、纯爱、言情、玄幻、悬疑、都市、仙侠8种主流小说文风

### 创作体验优化
- **历史记录管理**: 支持50步撤销/重做操作，创作过程随时回溯
- **本地自动保存**: 编辑内容实时保存到浏览器本地存储，刷新页面不丢失
- **双创作模式**: 内置V模式（均衡稳定）、O模式（高创意发散）双模式切换
- **自定义字数控制**: 支持100/200/300/500/1000字预设，也可自定义50-5000字目标字数
- **沉浸式悬浮窗**: 独立悬浮窗编辑器，零污染主页面，支持Esc一键关闭
- **全场景适配**: 完美适配深色模式、PC端与移动端响应式布局

### 稳定性与兼容性
- **破限防空回机制**: 内置3次自动重试，拦截空内容、拒绝生成提示，确保稳定输出有效内容
- **API限流保护**: 每分钟最多10次API调用，避免触发后端限流规则
- **参数继承支持**: 可选择继承SillyTavern全局AI生成参数，适配所有可用的AI后端
- **纯净请求保障**: 彻底修复UI内容混入API请求的问题，仅发送用户编辑的小说正文

## 🚀 安装教程
### 标准安装方式（推荐）
1. 打开你的SillyTavern服务，进入顶部「扩展」页面
2. 点击「安装扩展」按钮，选择「从URL安装」选项
3. 在输入框中填入项目仓库地址：
 
 
https://github.com/Neohero521/Continuation_machine
 
plaintext  
4. 点击「安装」按钮，等待安装完成
5. 安装成功后刷新SillyTavern页面，即可在扩展列表找到插件

### 手动安装方式
1. 下载项目完整代码压缩包，解压到本地
2. 将解压后的文件夹完整复制到SillyTavern的 `public/scripts/extensions/third-party` 目录下
3. 重启SillyTavern服务，刷新页面即可启用

## 📖 使用说明
### 基础创作流程
1. **打开编辑器**: 在SillyTavern扩展面板，找到「彩云小梦复刻版」，点击「打开彩云小梦编辑器」按钮，弹出悬浮窗编辑器
2. **输入原文**: 在编辑区输入你的小说正文，将光标定位到需要续写/修改的位置
3. **选择创作功能**: 点击左下角五角星按钮，打开功能菜单，选择需要的创作功能（续写/扩写/缩写/改写/定向续写）
4. **选择文风**: 点击底部风格选择按钮，打开文风菜单，选择适配你小说的创作风格
5. **配置字数**: 点击右上角齿轮图标，打开设置面板，设置单条内容的目标字数
6. **生成内容**: 点击右下角「Ai 继续」按钮，等待AI生成内容
7. **选择与编辑**: 生成完成后，在底部结果区点击不同分支预览内容，支持直接修改续写内容，可选择「保存」「撤回」「继续续写」，不满意可点击「换一批」重新生成

### 功能说明
| 功能 | 使用要求 | 功能说明 |
|------|----------|----------|
| 续写 | 无额外要求 | 从光标位置开始，续写后续剧情内容 |
| 扩写 | 需先选中要扩写的文本 | 丰富选中内容的细节、场景、心理描写 |
| 缩写 | 需先选中要缩写的文本 | 精简选中内容，保留核心剧情与信息 |
| 改写 | 需先选中要改写的文本 | 用选中的文风重写内容，不改变核心情节 |
| 定向续写 | 无额外要求 | 可在输入框填写自定义要求，按指定方向续写 |

## ⚙️ 配置说明
1. **继承ST全局AI生成参数**
- 开启后：插件将完全使用SillyTavern全局设置的AI生成参数（温度、top_p、重复惩罚、最大生成长度等）
- 关闭后：使用插件内置的默认参数，适配小说创作场景
2. **V模式 / O模式切换**
- V模式：均衡稳定模式，默认参数 `temperature=0.7` `top_p=0.85`，适合常规剧情续写
- O模式：高创意模式，默认参数 `temperature=1.0` `top_p=0.95`，适合脑洞大开的剧情创作
3. **续写字数设置**
- 预设选项：100字、200字、300字、500字、1000字
- 自定义范围：50-5000字，可根据创作需求自由设置

## ⚠️ 注意事项
1. 插件需要SillyTavern已配置可用的AI后端（支持本地大模型、OpenAI/Azure API、Anthropic API等所有SillyTavern兼容的后端）
2. 生成内容的质量、流畅度、风格匹配度，取决于你使用的AI模型，建议使用经过小说创作优化的大模型
3. 插件会将编辑内容自动保存到浏览器本地存储，清除浏览器缓存/数据会导致未导出的内容丢失，重要内容请及时备份
4. 请遵守国家相关法律法规与AI生成内容规范，插件仅用于合法的虚构文学创作，禁止用于生成违规内容
